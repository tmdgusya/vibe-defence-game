# Defence Garden - Asset Generation Prompts

## Image Generation Guidelines

- **Style**: Vibrant cartoon/comic style, similar to Plants vs Zombies
- **Format**: PNG with transparent background
- **Art Style**: Clean 2D cartoon or pixel art with bold outlines
- **Color Palette**: Bright, saturated colors with high contrast
- **Lighting**: Top-down lighting with slight shadows for depth

## ðŸ“‹ Batch Generation Tips

1. **Use consistent art style** across all assets (same artist/generator)
2. **Maintain color consistency** within each category (enemies, towers)
3. **Test in-game** - import one asset first to verify size/style
4. **Export settings**:
   - Format: PNG
   - Background: Transparent
   - Color mode: RGBA
   - DPI: 72 (web standard)

5. **File naming convention**:

   ```
   Enemies: enemy-basic.png, enemy-tank.png, enemy-flying.png, etc.
   Towers: tower-sunflower-1.png, tower-sunflower-2.png, tower-sunflower-3.png, etc.
   Projectiles: projectile-pea.png, projectile-mortar.png
   ```

6. **Save location**:
   ```
   /public/assets/enemies/
   /public/assets/towers/
   /public/assets/projectiles/
   ```
